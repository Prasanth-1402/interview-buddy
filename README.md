
# AI Interview Buddy

An intelligent, interactive mock interview application powered by **Google Gemini** (Brain) and **Vapi** (Voice). This application simulates real-time technical interviews with dynamic personalities, feedback analysis, and session tracking.

## Features

- **Real-time Voice Interaction**: Converse naturally with the AI interviewer using Vapi.ai.
- **Dynamic AI Personas**: Interviewer varies between roles like "Socratic", "Direct", "Deep Dive", etc.
- **Live Feedback**: Get instant analysis on your answers.
- **Detailed Reports**: Post-interview feedback scores, strengths, weaknesses, and improvement tips generated by Gemini 2.5 Flash.
- **Session History**: Track your progress with a local history of all your interviews.
- **Responsive UI**: optimized for all devices with a modern, dark-themed interface.

## Prerequisites

- Node.js 18+ installed.
- A [Google AI Studio](https://aistudio.google.com/) account (for Gemini API).
- A [Vapi.ai](https://vapi.ai/) account (for Voice AI).

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Start-Up-Simplified/interview-buddy.git
cd interview-buddy
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Environment Setup

Create a `.env.local` file in the root directory:

```bash
cp .env.example .env.local
# Or create it manually
touch .env.local
```

Add the following environment variables to `.env.local`:

```env
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
NEXT_PUBLIC_VAPI_PUBLIC_KEY=your_vapi_public_key_here
NEXT_PUBLIC_VAPI_ASSISTANT_ID=your_vapi_assistant_id_here
```

### 4. How to Get API Keys

#### **Google Gemini API Key**
1.  Go to [Google AI Studio](https://aistudio.google.com/).
2.  Click on **"Get API key"** in the sidebar.
3.  Click **"Create API key"** (you can create one in a new project).
4.  Copy the key and paste it as `NEXT_PUBLIC_GEMINI_API_KEY`.

#### **Vapi API Keys**
1.  Sign up/Login to [Vapi.ai](https://dashboard.vapi.ai/).
2.  **Public Key**:
    *   Go to the **API Keys** section in the dashboard (usually under Settings or Account).
    *   Copy your **Public Key** and paste it as `NEXT_PUBLIC_VAPI_PUBLIC_KEY`.
3.  **Assistant ID**:
    *   Go to the **Assistants** tab.
    *   Create a new Assistant (or use an existing one).
    *   *Tip: Use a "Functions" capable assistant to allow potential future tool calling.*
    *   Copy the **Assistant ID** (it looks like a UUID) and paste it as `NEXT_PUBLIC_VAPI_ASSISTANT_ID`.

### 5. Run the Application

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start practicing!

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **AI Brain**: Google Gemini 2.5 Flash / 3.0 Pro (via SDK)
- **Voice**: Vapi.ai Web SDK
- **Icons**: Lucide React

## Project Structure

- `app/`: Next.js App Router pages and layouts.
- `Components/`: Reusable UI components (InterviewSession, Dashboard, etc.).
- `lib/`: Utility functions (Gemini client, storage, etc.).
- `lib/storage.ts`: LocalStorage manager for interview history.
